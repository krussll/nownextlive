# STYLE_GUIDE.md

## Purpose
Defines visual and structural style guidelines for NowNext documentation, website, and UI.

## Typography
- **Headings:** Short, clear, sentence case.
- **H1:** 32–40px (bold)
- **H2:** 24–28px (semi-bold)
- **H3:** 18–20px (medium)
- **Body:** 16px, regular weight.
- **Font Style:** Clean, sans-serif (e.g. Inter, Roboto, Open Sans).

## Colour Palette
- **Primary Blue:** #2A66F0 (actions, links)
- **Dark Navy:** #1B2330 (headers, backgrounds)
- **Soft Grey:** #F4F6F8 (backgrounds)
- **White:** #FFFFFF (content backgrounds)
- **Accent Green:** #3ECF8E (success states)

## Components
### Buttons
- Primary: Blue background, white text.
- Hover: Slightly darker blue.
- Corners: 6–8px radius.
- Text: Short verbs (“Update screen”, “Try demo”).

### Cards / Sections
- White background, soft shadow.
- 16–24px padding.
- Rounded corners (8px).

## Layout
- Use generous spacing.
- Avoid clutter.
- Align content in a single column on mobile, grid on desktop.

## Imagery
- Simple line illustrations.
- Light, minimal, clean.
